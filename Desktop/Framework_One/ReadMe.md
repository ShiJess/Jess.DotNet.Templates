## Framework版本

* 4.0 —— 针对企业开发时，由于部分企业内部软硬件环境限制，4.0是比较合理的版本

## 涉及技术

* WPF —— UI呈现
  * MVVM
* DI/IOC —— 注入容器
  * Autofac
* EF —— 数据持久化
  * 示例中使用SQLite做持久化
* Wix Toolset—— 安装包制作
* 单元测试
  * xunit
